AI-powered social media analytics dashboard for Kellermann GmbH, built with Python, Streamlit, and Pandas.

# 🏍️ Kellermann AI Social Media Command Center

An enterprise-grade analytics dashboard designed to track social media performance and AI content ROI.

## 🚀 Key Features
- **AI Executive Summary:** Dynamic, data-driven marketing recommendations.
- **A/B Testing:** Comparison of AI-generated vs. Human-generated content performance.
- **Interactive Filtering:** Drill down by Date, Platform, and Product (Atto, Jetstream, etc.).
- **Premium UI:** Dark-themed, responsive interface designed for high-end brand management.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Streamlit (UI), Pandas (Data Engine), Plotly (Interactive Charts)
- **AI Tooling:** Built using advanced prompting in Cursor AI.

## 🏃 How to Run
1. Clone the repo.
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
